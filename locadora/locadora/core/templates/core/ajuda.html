Como usar a aplicação?

Após Cadastrar um usuario para acessar a aplicação tem se acesso ao página inicial, nesta página tem-se acesso a barra de opções que o leva para a opção desejada de cadastrado de acordo com o respectivo nome. Esta aplicação tem como objetivo a gerencia básica para uma locadora de veiculos que precisa registrar os seus dados sobrel locações de veiculos funcionarios etc. Possui as seguintes funcionalidades para cadastro,
1. Menu de Login para que apenas usuarios cadastrados tenham acesso
2. Opções para cadastros diversas
2.1 Cadastro de Veiculos Disponiveis para Alugar
2.2 Cadastro de Locações Realizadas
2.3 Cadastro de Empregados da locadora
2.4 Cadastro de Respectivas funções para Empregados para fins de pagamento.
2.5 Cadastro de Clientes
3. Página de ajuda Voltar
